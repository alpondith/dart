# Basic

### Hello World!

```
void main(){
  print("Hello World!");
}
```

{% hint style="success" %}
Hello World!
{% endhint %}

### Print Name

```
void main(){
  print("Mr. Jhon");
}
```

{% hint style="success" %}
Mr. Jhon
{% endhint %}

### Print Multiple Names

```
void main(){
  
  print("Ava");
  print("Sophia");
  print("Emily");
  print("Olivia");
  print("Layla");
  print("Grace");
  
}
```

{% hint style="success" %}
Ava&#x20;

Sophia

Emily&#x20;

Olivia&#x20;

Layla&#x20;
