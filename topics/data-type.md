# Data Type

```
void main(){
  
  int a , b ;
  String name, email;
  
  a = 100;
  b = 200;
  name = "Mr Jhon";
  email = "user@gmail.com";
  
  print("Informations : $a , $b , $name , $email ");
  
}
```

{% hint style="success" %}
Informations : 100 , 200 , Mr Jhon , user@gmail.com
{% endhint %}
