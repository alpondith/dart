# Variable

```
void main(){
  
  var number = 5; 
  var num = 56.7;
  var message = "hello world";
  var name = "Mr Tanvir";
  var statement = true;
  
  print(number);
  print(num);
  print(message);
  print(name);
  print(statement);
  
}
```

{% hint style="success" %}
5&#x20;

56.7&#x20;

hello world&#x20;

Mr Tanvir&#x20;

true
{% endhint %}

```
void main(){
```
